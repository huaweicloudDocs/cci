# 列出所有batch/v1版本的API<a name="cci_02_3121"></a>

## 功能介绍<a name="section40622221"></a>

This API is used to list APIResources of GroupVersion "batch/v1".

## URI<a name="section30055677"></a>

GET /apis/batch/v1

## 请求消息<a name="section2065644"></a>
